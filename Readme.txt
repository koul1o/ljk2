-To run the platform launch the run.bat file for Windows or run.sh for Linux.
-To change the experimental settings edit the files using a text editor by altering the following: 
	fullscreen= y or n, to project in fullscreen
	tTime= integer, to set the duration of the training
	fTime= integer, to set the duration of the final quiz 
	step=  integer, to set the progress bar step  
	root= name_of_folder_containing_the_content
	experimentId= alphanumeric, to set the id of the experiment
	highlightEnabled= true or false, to enable higjlighting
 
-If errors occur please verify that the "bin" folder exists and contains the files found in the "src" folder, if not copy the contents of "src" to "bin".
